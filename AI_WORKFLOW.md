# AI Development Workflow

This repository uses an AI-assisted development workflow.

## Workflow

Idea → Feature → Implementation → Tests → Review

The development process uses specialized AI agents.

---

## Agents

### Domain Agent

Responsible for maintaining domain consistency.

Prompt:

ai/prompts/domain_agent.md

---

### Feature Architect

Converts product ideas into structured features.

Output location:

docs/features/

---

### Developer Agent

Implements the feature.

---

### Tester Agent

Creates tests validating the feature.

---

## Typical Development Flow

1. Write an idea
2. Generate a feature
3. Implement feature
4. Generate tests
5. Review architecture
